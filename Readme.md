# Workshop - Creando un Chat Bot con .NET 8 y Azure OpenAI

Bienvenido al taller dónde aprenderemos a crear un Chat Bot con Blazor WebAssembly en .NET 8 y Azure OpenAI

* SDK .NET 8
* Visual Studio Code
